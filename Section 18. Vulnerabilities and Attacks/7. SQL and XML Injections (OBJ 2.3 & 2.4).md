## XML Injection

### What Is XML?

**XML (Extensible Markup Language)** is used for:

* Data exchange
* Authentication & authorization
* Application-to-application communication

XML is:

* Human readable
* Tag-based
* Customizable (unlike HTML)

---

### What Is XML Injection?

An **XML injection** occurs when an attacker sends **malicious XML data** that is processed by an application.

If XML input is not validated, attackers can:

* Read local files
* Cause denial of service
* Manipulate application logic

---
