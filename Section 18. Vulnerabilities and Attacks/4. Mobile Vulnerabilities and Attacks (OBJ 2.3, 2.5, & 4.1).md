## 3. Insecure Connection Methods

Mobile devices rely heavily on **wireless connectivity**, which increases risk.

### High-risk connections:

* Open or unsecured Wi-Fi networks
* Unknown Bluetooth devices
* Discoverable Bluetooth mode

### Threats:

* Eavesdropping
* On-path (MITM) attacks
* Unauthorized data access

### Mitigation:

* Prefer **cellular data** over public Wi-Fi
* Use secured Wi-Fi (strong passwords or **802.1X**)
* Pair Bluetooth only with trusted devices
* Disable Bluetooth discoverability when not pairing

ðŸ“Œ *Exam tip:* Open Wi-Fi + Bluetooth = **attack surface**

## 4. Mobile Device Management (MDM)

**Centralized security control for mobile devices**

### Key Functions of MDM:

* **Patching**: Pushes OS and security updates
* **Configuration management**: Enforces security baselines
* **Policy enforcement**:

  * Disable sideloading
  * Detect jailbroken/rooted devices
  * Force VPN usage
  * Enforce encryption and screen locks

### Benefits:

* Reduces mobile attack surface
* Ensures compliance with security standards
* Centralized control for enterprise devices
