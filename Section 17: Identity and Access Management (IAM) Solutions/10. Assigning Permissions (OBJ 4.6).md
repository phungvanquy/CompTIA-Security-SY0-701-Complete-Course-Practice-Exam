## What Is Access Control?

**Access control** determines **who can access what** on a computer system and **what actions they can perform**.

* Users prove identity through **authentication**
* Permissions define **allowed actions**
* Goal: **Protect system integrity and data**

## User Account Control (UAC)

### What Is UAC?

A **security feature in Windows** that:

* Requires approval for admin-level actions
* Helps enforce least privilege

### How It Works

* Prompts for:

  * Admin password, or
  * Confirmation (Yes/No)
* Triggered when:

  * Installing software
  * Changing system settings

ğŸ“Œ Shield icon = UAC prompt required
